  • Adding Sound to the Animation
    • After being relatively satisfied with my sketch, I began to add sound to accompany with the movements; I refered to the P5js example online for reference and learned that you could load a file for sound playing. For adding a sound that suits my sketch, I first approached to search sound on YouTube and was trying to convert them into MP3 file. Almost all the YouTube videos I found are very long and made the progress challenging. Recalling from previous coding classes, I remembered that there were free resources online for coding. I explored to a new website where it lists a lot of free sound assets for use. Now that finding a sound fits my sketch, I started to implement it. I do encountered more problems when putting it in.
    • After setting up the variable and load the sound MP3 file, I imagine that my sound will repeat each time the color animation plays. I put sound.play() in the draw() function. As soon as I put it in there, all the pattern I have sketched disappeared and no sound was playing when I ran the code. Being confused, I tried to move the code into Setup() function, and found the same result as well. This is confusing for me because I assumed that, it is reasonable for Draw() to stop working because it is running repeatively and could be stucked by keep calling the play function; however, if you put the line of code into Setup() function, it should play automatically when you run the function right?
    • With trails and errors, I looked into the P5js music playing example again, and used the mousePressed() function to make the sound playing based on whether the mouse is clicked at the sketch. I looked further into documentations and manuals for sound and learned that sound playing is usually structured this way for functioning. Therefore, although different than I've imagined, a sound will play only when the mouse is clicked at my sketch. That being solved, now copying and pasting my work from P5js editing page to sketch.js, my sound code does not work and makes my sketch disappear on html page again. After spending quite sometime troubleshooting, I am officially stucked and will ask Professor Maxim Safioulline for help during the class.
    • After talking to Professor Maxim Safioulline, I learned that it was restricted on purpose, so that way users will not be bothered by sound ads or trying to find a sound playing among the tabs. We can usually solve this problem by implementing a button for user to click, so that everything started running automatically once the button is clicked, including the sound. I found this fact facinating and was happy to learn the reason of this issue and how I can navigate through it.
